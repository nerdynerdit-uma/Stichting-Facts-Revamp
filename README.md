# Stichting Facts Website

Website for Stichting Facts, featuring project information, gallery, team, and contact details.

## Project Structure

- Static HTML website
- Responsive design
- Multiple pages including:
  - Home page
  - About Us (Over Ons)
  - Projects (Projecten)
  - Gallery
  - Contact
  - News (Nieuws)
  - Team information
  - RVT (Raad van Toezicht)

## Deployment

This project is ready to be deployed on Vercel. Simply connect your GitHub repository to Vercel for automatic deployments.
